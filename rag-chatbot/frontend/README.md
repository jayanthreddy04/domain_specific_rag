# RAG Chatbot Frontend

React chat interface for the AI Knowledge Assistant.

## Setup

```bash
cp .env.example .env
npm install
npm start
```

Set `REACT_APP_API_URL=http://localhost:5001/api/chat` if the backend uses a different port.

## Features

- Responsive dark-theme chat UI
- Session-based conversational memory
- Expandable source citations with relevance scores
- Suggested starter questions
- Knowledge base status indicator
